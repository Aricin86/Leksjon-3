## Bash cardio
```javascript
// Lage mappe
$ mkdir folder

// Lage fil i mappen
$ cd folder
$ touch file.txt

// Legge tekst til filen
$ echo "Hi there..." > file.txt

// Endre den fra kommandolinjen
$ echo "Another line?" >> file.txt

// Slette filen
$ rm file.txt
```
